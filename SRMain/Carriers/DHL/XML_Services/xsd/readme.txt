Note for developer Usage
========================

A) datatypes_global XSD:
========================
1) datatypes_global_v10.xsd = XML Global Schema version 10, XML Schema version 10 usage only.
2) datatypes_global_v62.xsd = XML Global Schema version 6.2 for XML Schema Pickup, Routing and Capability and Quote Service usage only.


C) Shipment Validation Service XSD:
===================================
1) ship-val-global-req-10.0.xsd = XML Global Schema version 10.0 Shipment Validation service request schema
2) ship-val-global-res-10.0.xsd = XML Global Schema version 10.0 Shipment Validation service response schema
3) ship-val-err-res.xsd = XML Global Schema Shipment Validation service error response schema

D) Pickup Service XSD:
======================
1) pickupdatatypes_global-3.0.xsd = XML Global Schema version 6.2 Pickup service data types
2) book-pickup-global-req-3.0.xsd = XML Global Schema version 6.2 Book Pickup service request schema
3) book-pickup-global-res-3.0.xsd = XML Global Schema version 6.2 Book Pickup service response schema
4) modify-pickup-global-req-3.0.xsd = XML Global Schema version 6.2 Modify Pickup service request schema
5) modify-pickup-global-res-3.0.xsd = XML Global Schema version 6.2 Modify Pickup service response schema
6) cancel-pickup-global-req-3.0.xsd = XML Global Schema version 6.2 Cancel Pickup service request schema
7) pickup-err-res.xsd = Pickup Error Response schema
8) datatypes_global_v62b.xsd = XML Global Schema datatypes for XML Schema Book and Modify Pickup v3.0 request schema


E) Capability and Quote Service XSD:
====================================
1) DCTRequestdatatypes_global.xsd = XML Global Schema version 6.1 Capability and Quote service request - data types schema
2) DCTResponsedatatypes_global.xsd = XML Global Schema version 6.1 Capability and Quote service response - data types schema
3) DCT-Response_global.xsd = XML Global Schema version 6.1 Capability and Quote service response schema
4) DCT-req_global-2.0.xsd = XML Global Schema version 6.2 Capability and Quote service request schema
5) DCT-Response_global-2.0.xsd = XML Global Schema version 6.2 Capability and Quote service response schema

F) Routing Service XSD:
=======================
1) routing-global-req-2.0.xsd = XML Global Schema version 6.2 Routing request schema
2) routing-global-res.xsd = XML Global Schema Routing response schema
3) routing-err-res.xsd = Routing Error Response schema

G) Tracking Service XSD:
========================
1) datatypes.xsd = XML Global Schema version datatypes for XML Schema Tracking usage only.
2) TrackingRequestKnown-1.0.xsd = XML Global Schema version 6.2 Known Tracking request schema
3) TrackingRequestUnknown-1.0.xsd = XML Global Schema version 6.2 Unknown Tracking request schema
4) TrackingResponse.xsd = Known/Unkown Tracking response schema
5) track-err-res.xsd = Tracking Error Response schema
